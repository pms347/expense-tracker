Expense Tracker

Connect your cloud database

Your data will be stored in Supabase — a free database. Once connected, your expenses load on any browser, any device, even incognito. Setup takes 3 minutes.

① Create free Supabase project

Go to supabase.com → Sign up → New Project → wait ~2 min for it to start.

② Run this SQL (one time only)

In your project → SQL Editor → paste and run:

create table if not exists expenses (id bigint primary key generated always as identity, name text, amt numeric, cat text, date text, note text, created_at timestamptz default now()); create table if not exists categories (id bigint primary key generated always as identity, name text unique, color text); alter table expenses disable row level security; alter table categories disable row level security;

③ Paste your credentials below

Project Settings → API → copy Project URL and anon public key.
