🌐 Cross-Device Sync (Supabase)

By default your data is saved in this browser only. To access your data from any device or browser, connect a free Supabase database. It takes 5 minutes and is completely free.

1Go to supabase.com → Sign up free → Create a new project
2In your project, open SQL Editor and run the SQL below to create the tables
3Go to Project Settings → API → copy your Project URL and anon public key
4Paste them below and click Connect
Run this SQL in Supabase SQL Editor:

create table expenses (id bigint primary key generated always as identity, name text, amt numeric, cat text, date text, note text, created_at timestamptz default now());

create table categories (id bigint primary key generated always as identity, name text unique, color text);
✅ Connected to Supabase! Your data syncs across all devices.
